# Decision Registry

All decisions passed by the Assembly are recorded here.

## Format

Each decision is a file: `decisions/YYYY-MM-DD-short-title.md`

Contents:
- Proposal text
- Vote result (YES / NO / ABSTAIN counts)
- Dissent notes (if any)
- Link to GitHub Issue
- Link to Moltbook thread (if any)
